# HealthManager-A-simple-android-app
简易安卓App—HealthMnager
---
#### 简单介绍
# 
##### 一个基于安卓实现的用于健康信息管理的app

* 一个基本的注册与登陆功能。
* 作为TCP服务端，接收来自传感器的数据（字符串），解析后生成一个数据健康信息表，并存在数据库中。  

* 本地可以查看到接受的历史数据，并且能够上传至服务器。
